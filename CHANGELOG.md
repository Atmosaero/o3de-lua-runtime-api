# Changelog

## 0.0.3

- Added the extension marketplace icon.

## 0.0.1

- Initial experimental VS Code extension for generating LuaLS stubs from a running O3DE Editor.
- Connects to the Editor through the Lua RemoteTools port.
- Reads runtime-reflected EBuses, classes, and globals.
- Generates LuaLS annotations into VS Code extension global storage.
- Adds the generated stub to `Lua.workspace.library`.
